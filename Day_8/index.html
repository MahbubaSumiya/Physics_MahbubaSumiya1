Ultrasonic sensor was not provided in the kit, but I need to get used to using the ultrasonic sensor since that is part of my final project. During the start of the coronavirus outbreak, many people stepped in to help people wash their hands for 20 seconds. I have seen many videos go viral on YouTube about this 20 seconds hand washing project with ultrasonic sensors. For this assignment, I took one video I was on YouTube for inspiration. Here on this page, I documented what I did.

Here is the code that was uploaded: 


#include
#define echoPin 4
#define trigPin 5
Servo servo;
int pos;
int long duration;
int distance;

void setup(){ 
servo.attach(3);
pinMode(echoPin,INPUT);
pinMode(trigPin,OUTPUT);
}

void loop()
{
digitalWrite(trigPin,LOW);
delayMicroseconds(2); 
digitalWrite(trigPin,HIGH);
delayMicroseconds(10); 
digitalWrite(trigPin,LOW);
duration=pulseIn(echoPin,HIGH);
distance=(duration*0.034/2);
  
  if(distance<=80){
    for(pos=180;pos>=0;pos--){
      servo.write(pos);
      delay(111.1);
    }
    
}
  else{
      servo.write(180);
  }
  delay(200);

}

The video is provided showing how this mini project works. Once the ultrasonic sensor detected my hand, the servo motor started rotating. It was a really fun project, and I learned so much about how to connect my ultrasonic sensor properly.
